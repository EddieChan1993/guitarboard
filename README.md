# guitarboard
吉他指板音名记忆训练
# 操作说明
## 模式切换
1 -简单模式
2 -进阶模式
3 -自由模式
## 现实切换
C -音名和唱名切换
## 显示切换
S -显示
H -隐藏
## 自由模式下标记主音
control+ 鼠标点击任意音名
标记主音
